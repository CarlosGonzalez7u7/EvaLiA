¿Cómo configuro las Rúbricas?

Para que EvaLiA automatice tus calificaciones, necesitas establecer las reglas del juego:

  • Configuración por Periodo: Las rúbricas que agregues (Ej. Tareas 40%) se aplicarán al periodo que esté ACTIVO.
  • Flexibilidad: Cuando termine el periodo actual, podrás decidir si mantienes los mismos porcentajes para el siguiente o si creas una rúbrica completamente nueva.
  • Regla de Oro: La barra de progreso debe llegar obligatoriamente a 100% en cada periodo para que el sistema pueda calcular promedios. Usa para borrar si te equivocas.
